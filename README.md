# AI SaaS Landing Page

A production-ready **Next.js** landing page template for modern AI/SaaS products — fast, responsive, and designed for conversion.

> **Goal:** ship a polished marketing site in hours, not days.

---

## Why this repo exists

Most landing-page starters are either:
- too bare (you still need to build everything), or
- too bloated (hard to edit, hard to deploy).

This project aims to be the “sweet spot”: clean structure, opinionated defaults, and easy customization.

---

## Highlights

- **Next.js App Router** architecture
- **Responsive UI** out of the box
- **SEO-friendly** structure (metadata-ready)
- **Fast dev experience** (hot reload, simple layout)
- **Deploy in minutes** with Vercel

---

## Quick start

### 1) Install

```bash
git clone https://github.com/Ali-Tamerr/ai-saas-landing-page.git
cd ai-saas-landing-page
npm install
```

### 2) Run

```bash
npm run dev
```

Then open: http://localhost:3000

---

## Project structure

> (Exact structure may differ slightly depending on your customizations.)

- `app/` — routes, layouts, metadata
  - `page.tsx` — main landing page
  - `layout.tsx` — shared layout wrapper
- `public/` — static assets

---

## Customize

Typical edits you’ll make first:

- **Hero + sections**: `app/page.tsx`
- **Global styles**: `app/globals.css`
- **Site metadata**: `app/layout.tsx`

## Deployment

### Deploy on Vercel (recommended)

1. Push your repo to GitHub
2. Import into Vercel
3. Deploy

Learn more: https://nextjs.org/docs/app/building-your-application/deploying
